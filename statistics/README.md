# Statistics

## Overview

This folder contains statistical analysis tools and scripts for processing and analyzing data sets. It provides implementations for calculating common statistical measures and data analysis operations.

## Features

- Average age calculation from author datasets
- Housing data analysis including mean, median, and mode calculations
- CSV data processing for statistical operations
- Rental price analysis for specific regions
