# Algorithms

## Overview

This folder contains a collection of algorithm implementations and demonstrations covering various problem-solving techniques and computational approaches. Each subfolder focuses on a specific algorithm category.

## Features

- Binary Search implementations for efficient sorted array searching
- Brute Force approaches to problem-solving
- Dijkstra's Algorithm for shortest path finding in weighted graphs
- Naive Pattern Search for basic string pattern matching
- Proof of Work demonstration for blockchain concepts
- Recursive algorithm implementations and examples
- Various sorting algorithms including bubble sort, merge sort, and quicksort

## Functions

Each subfolder contains its own detailed documentation. Refer to individual subfolder README files for specific function descriptions.

## Usage

Explore each subfolder to view the implementation and usage examples for each algorithm.
