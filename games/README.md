# Games

## Overview

This folder contains a collection of interactive game implementations and text-based adventures in Python. From casino-style betting games to classic puzzles and arena combat, these projects demonstrate game logic and interactive programming.

## Features

- Dice betting game with username creation and money management
- Gladiators arena combat game with attack and defense mechanics
- Number guessing game with hints and score tracking
- Scrabble word game implementation
- Towers of Hanoi puzzle with recursive solution
- Multiple game progression systems including weapon leveling and character stats

## Functions

Each subfolder contains its own detailed documentation. Refer to individual subfolder README files for specific function descriptions.

## Usage

Navigate to each game's folder and run the main script to play. Each game has its own instructions and mechanics.
