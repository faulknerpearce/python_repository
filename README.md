# Python Projects

## Overview

This repository contains a variety of Python scripts and programs for educational and demonstration purposes. It covers a wide range of topics from algorithms and data structures to games, file processing, and statistical analysis.

## Contents

### Algorithms
Implementations and demonstrations of a variety of algorithms including binary search, sorting algorithms, pathfinding algorithms, and more.

### Computer Architecture
Explorations of fundamental computer architecture concepts including logic gates and digital circuit implementations.

### Data Structures
Examples and implementations of fundamental data structures such as HashMaps, Stacks, Queues, and Linked Lists.

### Games
A collection of interactive game implementations including dice betting games, combat games, guessing games, and classic puzzles.

### Profile Management
Python code related to user account management and profile handling, including login systems and authentication.

### Projects
Larger Python projects demonstrating real-world applications including escape rooms, recommendation software, and routing programs.

### Read Files
Utilities and examples for reading, parsing, and processing various file formats including CSV, JSON, and text files.

### Sandbox
This folder is for miscellaneous Python scripts and small projects.

### Statistics
Statistical analysis tools and scripts for processing and analyzing data sets, including housing data analysis.

### Trees and Graphs
Implementations of fundamental tree and graph data structures along with associated algorithms for traversal and pathfinding.

## Usage

Navigate to any folder to explore the implementations. Each folder contains its own README with detailed information about the project.
