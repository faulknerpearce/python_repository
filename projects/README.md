# Projects

## Overview

This folder contains a collection of larger Python projects demonstrating various real-world applications and complex system implementations. These projects showcase more advanced programming concepts and complete system designs.

## Features

- Escape Room: Interactive text-based adventure game
- Gladiators Game: Combat simulation with character progression
- Recommendation Software: Item suggestion based on user preferences
- Routing Program: Pathfinding algorithm implementation
- Sorted Bookshelf: Book collection organization application
- Traveling Salesperson: Algorithms for solving the TSP

## Functions

Each subfolder contains its own detailed documentation. Refer to individual subfolder README files for specific function descriptions.

## Usage

Navigate to each project's folder to explore the implementation and run the main script to interact with the project.
