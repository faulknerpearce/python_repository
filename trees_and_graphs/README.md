# Trees and Graphs

## Overview

This folder contains implementations of fundamental tree and graph data structures along with associated algorithms. These data structures are essential for solving complex computational problems and are widely used in software development.

## Features

- Graph implementations including adjacency list representation
- Tree data structures including binary trees and binary search trees
- Graph traversal algorithms (BFS, DFS, A*, Dijkstra's)
- Tree traversal methods (in-order, pre-order, post-order)
- Heapsort implementation using max heap
- Pathfinding algorithms for weighted graphs

## Functions

Each subfolder contains its own detailed documentation. Refer to individual subfolder README files for specific function descriptions.

## Usage

Navigate to the graphs or trees subfolder to explore specific implementations. Each folder contains detailed README files with usage instructions.
